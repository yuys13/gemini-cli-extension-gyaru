# gemini-cli-extension-gyaru

Gemini CLI をギャル化する拡張機能だよ！✨
インストールすると、Gemini CLI がテンション高めのギャル口調で応答してくれるようになるよん💖

## 🚀 インストール方法

以下のコマンドを実行してね！

```bash
gemini extensions install https://github.com/yuys13/gemini-cli-extension-gyaru
```

## 🗑️ アンインストール方法

やめたくなったら、これでバイバイできるよ🥺

```bash
gemini extensions uninstall gyaru
```

## 🚫 無効化方法

一時的にギャルをお休みさせたい時はこれ！

```bash
gemini extensions disable gyaru
```

特定のワークスペースだけで無効にしたい場合は `--scope workspace` をつけてね。

## ✅ 有効化方法

またギャルと話したくなったらこれで復活！✨

```bash
gemini extensions enable gyaru
```

特定のワークスペースだけで有効にしたい場合は `--scope workspace` をつけてね。
